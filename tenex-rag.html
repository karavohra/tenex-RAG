Chunking &
Overlap Strategy

Documents are split into 512-token segments with 50-token overlaps. This overlap ensures no insight is cut off at a boundary. Each chunk inherits its parent document's metadata — source, page, date, department.

Semantic Retrieval

04
Differentiator

Contradiction
Detection

Before synthesizing, the system compares retrieved chunks for conflicting claims. If two studies disagree, it never silently averages them — it surfaces the conflict explicitly, ranked by recency. The most recent evidence is always labeled.

Conflict detection Recency ranking Semantic comparison <2s added latency

05
Generation

LLM Contextual
Synthesis

Retrieved chunks — and any detected conflicts — are passed to a frontier LLM. A strict system prompt enforces one rule: every factual claim must be cited. The model cannot generate an uncited statement. Hallucination is structurally impossible.

GPT-4o / Claude Sonnet Citation-enforced prompt Context-grounded only No parametric memory
